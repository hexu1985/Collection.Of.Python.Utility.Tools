正确打开方式——ANACONDA和jupyter安装

沫沫
461 人赞同了该文章

如果一个数据分析的教程介绍python的打开方式不是从安装anaconda和jupyter开始，那么对于初学者就意味着落入无限崩溃的深渊，python本身的命令窗口模式不支持可视化，学习和练习起来无比崩溃，各种IDE解释器也是五花八门，各有优劣。

anaconda自带的各种第三方工具能够免去后面学习中的麻烦，直接导入非常方便，jupyter更是精简轻便，可视化效果好，简单明了，又能进行注释，有利于理解语句和理清思维。把anaconda和jupyter的安装使用操作路径备份如下，方便自己也方便大家~

1.Anaconda的好处

2.安装步骤

3.如何管理包

4.Jupyter notebook的使用


1.Anaconda的好处

1）Anaconda 自带了一大批常用数据科学包，它附带了 conda、Python 和 150 多个科学包及其依赖项，所以后期使用的时候就不用一个一个数据包再去安装了
2）管理包
Anaconda 是在 conda（一个包管理器和环境管理器）上发展出来的。
在数据分析中，conda（包管理器）可以很好的帮助你在计算机上安装和管理第三方包，包括安装、卸载和更新包。
3）管理环境
Anaconda可以同时支持python2和python3等不同版本，不同项目在不同环境下运行。

形象一点说，Anaconda就相当于一个python的操作系统，自带了不同版本的python和第三方包以及其他相关软件应用，所以安装好Anaconda，就不需要再单独安装python了，可以直接在这个操作系统里进行添加和删除。


2.安装步骤

官网下载地址：https://www.continuum.io/downloads （选择相应版本就好，速度比较慢，可以用迅雷等第三方下载工具）

如果没有安装python，就可以不用再安装了，已经安装了也不影响，系统会默认使用Anaconda里的python

注意：如果你是windows 10系统，注意在安装Anaconda软件的时候，右击安装软件→选择以管理员的身份运行。

注意：这里提示下，安装过程中请把这个步骤的两个复选框都选上，特别是第一个，如果不勾上后面使用会报错，需要自己手动在windows系统里添加环境变量path，所以这里注意下，后面可以少很多坑

完成安装后，如果你是在windows上操作，按下面图打开 Anaconda Prompt （或者 Mac 下的终端），就是类似windows的命令窗口这个。

注意：如果你是windows 10系统，打开Anaconda Prompt请右键 →选择以管理员的身份运行。

可以在终端或命令提示符中键入 conda list，以查看你安装的内容。（注意：输入法用英文，半角）

啪啦啪啦一大串三方包就出来，这个时候是不是很满足于anaconda的强大？

接下来我们要更新下所有的三方包，免得后期调用的时候报错再一个一个更新就会麻烦很多。继续在这个anaconda的命令窗口里输入更新所有包的命令：

conda upgrade    --all

并在提示是否更新的时候，即上图proceed（[y]/n）?输入 y（Yes）让更新继续。然后你就看着一个一个包开始更新到进度100%。

注意：一般anaconda都会默认安装在c盘根目录下，所以大家的命令窗口打开后可以直接敲入命令就好，我比较作死，安装的时候选择了F盘建了个新文件夹anaconda，所以我的命令是上面截图的标题栏那个一长串的conda upgrade --all --prefix F:\anaconda anaconda 所以大家根据自己的实际情况来看，多看报错提示，多百度。


3.如何管理包

安装了 Anaconda 之后，以后我们就可以很方便的管理包了（安装，卸载，更新）。

1）安装包在命令窗口中键入：conda install package_name

eg: 要安装 pandas，在命令窗口中输入：conda install pandas

2）卸载包在命令窗口中键入 ：conda remove package_names

eg: conda remove pandas3）

3）更新包在命令窗口中键入：conda update package_name

最常用的pandas，和numpy都是已经安装好了的，不用再重复安装啦


4.Jupyter notebook的使用

Jupyter Notebook 现在差不多数据分析，机器学习的必备工具。因为它可以让数据分析师集中精力向用户解释整个分析过程，而且界面简单明了。

安装jupyter很简单，打开上面anaconda Prompt，对，还是上面那个命令窗口,输入命令：

conda install jupyter notebook


启动Jupyter notebook的两种方式：

一是在anaconda Prompt里输入命令：jupyter notebook

二是启动anaconda，如图所示

然后你的默认浏览器就会自动打开jupyter界面，对的，jupyter的操作使用界面是web形式，直接在网页上进行编辑使用。


安装环境自动关联包

conda install nb_conda

该包可以将conda中创建的环境自动关联到你的notebook中。

这时环境名称py3没有出现在notebook中。解决办法是按下图步骤安装包ipykernel。

那个旋转的小图标是最后点的哈，第五步。这个时候点回files，点击右上角new新建notebook的时候就可以看到py3的选项了，如果需要用到py2，可以重复上面的操作一次。

在正式使用jupyter之前，先在Anaconda命令窗口安装代码自动补全包

conda install pyreadline

这一步不要偷懒，可以解决后面在notebook里tab键不能补齐代码的问题。


jupyter简单使用

files界面，new新建，这里可以新建一个notebook（根据你需要的环境和python版本）、文本文件、文件夹或者终端。

我新建了一个文件夹，勾选后rename为learn python，点开learn python文件夹，创建一个基于py3的notebook


这个时候浏览器会自动打开一个名为Untitled的新标签页，

3是快捷键，可以快速保存，增加，剪切，换行，运行等操作

4是单元格，是编写和运行代码的地方

来，我们开始编程的第一步，在单元格里输入print('hello world!')，然后点击快捷键里的三角形run，开始迈开你用python编程的第一步吧，看看会有什么发生？o(∩_∩)o
